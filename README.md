# GaloDev Portfolio v2.0

Este es el proyecto oficial del portafolio profesional de Steven GaloCR, especializado en desarrollo Full Stack. 
Aquí expongo mis servicios, habilidades y un medio directo para contactarme ofreciendo soluciones de software a la medida de mis clientes.

## 🚀 Tecnologías Principales

- **[Next.js 15](https://nextjs.org/)** - App Router y Server Actions
- **[React 19](https://react.dev/)** - UI dinámica y robusta
- **[TypeScript](https://www.typescriptlang.org/)** - Tipado estricto para menor incidencia de errores
- **[Tailwind CSS](https://tailwindcss.com/)** - Diseño ágil, estético y responsive
- **[Framer Motion](https://www.framer.com/motion/)** - Animaciones fluidas, limpias y profesionales
- **[React Icons](https://react-icons.github.io/react-icons/)** - Iconografía moderna
- **[Nodemailer](https://nodemailer.com/)** - Sistema de mensajería (Formulario de contacto seguro)

## 🎯 Objetivo y Funcionalidades

- **Portafolio Moderno**: Exhibición de mis servicios, perfil y formas de trabajar.
- **Micro-Animaciones**: Experiencia de usuario inmersiva con efectos espaciales, glow y parallax.
- **Múltiples Idiomas (i18n)**: Soporte completo, preparado para abarcar clientes internacionales.
- **SEO Ready**: Etiquetas configuradas y metadatos implementados garantizando una correcta indexación por los motores de búsqueda (Google).
- **Formulario Integrado**: Comunicación directa segura usando API routes, validaciones anti-spam y SMTP propio para recepción a tu bandeja en tiempo real.

## ⚙️ Entorno de Desarrollo

### 1. Clona e instala dependencias
```bash
npm install
```

### 2. Configuración de Variables
Crea el archivo correspondiente a las variables de entorno `.env` en la raíz de tu proyecto usando `.env.example` como plantilla.

Asegúrate de configurar los parámetros para envío de formulario (ej. tu App Password de Gmail):
```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=tu_correo@gmail.com
SMTP_PASS=tu_clave_de_aplicacion
CONTACT_EMAIL=tu_correo_de_recepcion@gmail.com
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 3. ¡Levanta el Servidor!
```bash
npm run dev
```
Navega a [http://localhost:3000](http://localhost:3000) en tu navegador para interactuar con la aplicación en tu entorno local.
